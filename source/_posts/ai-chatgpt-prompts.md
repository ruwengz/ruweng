ChatGPT 已经成为我日常工作中不可或缺的助手。以下是我总结的一些高效 Prompt 技巧。

## 1. 明确角色设定

```
你是一位资深前端架构师，擅长 React 性能优化。请帮我分析以下代码的性能问题...
```

## 2. 分步骤提问

```
第一步：请解释这个概念
第二步：基于上述解释，写一个示例
第三步：指出常见错误
```

## 3. 提供上下文

```
我正在开发一个电商项目，使用 Vue 3 + TypeScript...
```

## 4. 指定输出格式

```
请用以下格式回答：
1. 优点
2. 缺点  
3. 使用场景
4. 代码示例
```

## 5. 要求解释思维过程

```
请一步步思考，并解释你的推理过程...
```

## 6. 设置约束条件

```
请用不超过 100 词解释，用类比的方式...
```

## 7. 让 AI 自检

```
请检查以下代码的潜在问题，并列出可能的风险...
```

## 8. 迭代优化

```
基于你的上一次回答，请优化/简化/详细解释...
```

## 9. 要求多种方案

```
给出 3 种实现方式，并比较优劣...
```

## 10. 让 AI 提问

```
请先问我一些必要的问题，以便给出更准确的答案...
```

## 结语

善用这些技巧，ChatGPT 能成为你更强大的生产力工具！ 🚀
