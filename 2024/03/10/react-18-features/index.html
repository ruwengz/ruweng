👨‍💻 开发者小明 👁 0 💬 true ⏱ 5 分钟

React 18 于 2022 年发布，带来了许多令人兴奋的新特性。本文将带你深入了解这些新功能。

Automatic Batching（自动批处理）

React 18 之前，只有 React 事件处理程序中的更新会被批处理。在 React 18 中，所有更新都会自动批处理，包括 Promises、setTimeout 等。
